# 🐾 Slime Click

> Кликай по слаймам, зарабатывай монеты, прокачивайся и открывай новые локации!  
> Легкий кликер, вдохновлённый вселенной **Slime Rancher 2**.

---

## ✨ Особенности
- 🖱️ **Кликер-механика** – простой и залипательный геймплей с элементами прогрессии
- 💰 **Система прокачки** – улучшайте урон за клик, скорость автокликера и бонусные множители
- 🗺️ **Локации** – на старте доступны 2 уровня. Каждая локация уникальна визуально и по балансу
- 📈 **Планируемые обновления** – игра активно развивается, контент будет добавляться регулярно
- 🎨 **Стилизация** – яркие спрайты и анимации, передающие атмосферу оригинала

---

## 🎮 Как играть
1. **Кликайте** по слаймам, чтобы получать монеты.
2. Откройте **Магазин** и покупайте улучшения:
   - 🔨 Урон за клик
   - 🤖 Автокликер
   - 💎 Критические удары и множители
3. Накопите нужную сумму, чтобы **разблокировать следующую локацию**.
4. Повторяйте цикл, прокачивайтесь и открывайте новый контент!
